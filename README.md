# 东方幻存神签 Web

一个非官方、非商业的东方 Project 同人网页抽签项目。

本项目将「东方幻存神签」整理为一个可直接在浏览器中使用的网页抽签体验。访问者可以点击抽签，查看随机出现的签文、角色信息、签运等级、分项运势和评价文字。

本项目不是官方作品，也不提供现实占卜或预测功能。它只是一个面向东方 Project 爱好者的网页展示与互动项目。

## 在线访问

如果本仓库已启用 GitHub Pages，可以通过项目页面直接访问：

```text
https://knotesgit.github.io/touhou_mikuji/
```

如果页面暂时无法打开，请以仓库当前的 Pages 设置为准。

## 功能

- 随机抽取一张幻存神签
- 显示签号、签运等级、角色名、称号和能力
- 使用竖排文字样式呈现主签文和详细签文
- 显示每张签对应的评价 / 评论内容
- 在浏览器本地保存最近抽签记录
- 支持从历史记录中重新查看已抽到的签
- 无需账号、无需后端服务器

## 使用方式

打开网页后，点击抽签按钮即可获得一张签。

抽签记录只保存在当前浏览器的本地存储中，不会上传到服务器。更换浏览器、清除浏览器数据或使用无痕模式时，历史记录可能会消失。

## 本地运行

这是一个静态网页项目，不需要安装复杂依赖。

可以直接打开 `index.html`，也可以在项目根目录启动一个本地静态服务器：

```bash
python -m http.server 8000
```

然后在浏览器中访问：

```text
http://localhost:8000
```

## 项目结构

```text
.
├── index.html              # 页面结构
├── css/
│   └── style.css           # 页面样式和签纸排版
├── js/
│   └── draw.js             # 抽签、渲染和历史记录逻辑
├── data/
│   ├── mikuji-index.json   # 签文数据索引
│   └── mikuji/             # 单张签文 JSON 数据
└── tools/
    └── import/             # 数据整理与导入辅助工具
```

普通访问者只需要打开网页使用即可；`tools/` 目录主要用于项目维护。
